# scene-story-agent

`scene-story-agent`는 사용자가 남긴 사진, 영상, 메모를 원본 기록으로 보관하고, AI 해석과 벡터 검색으로 기록 사이의 연결을 보여주는 멀티모달 AI 에이전트 프로젝트다.

상세한 제품 정의와 기술 선택은 아래 문서를 기준으로 한다.

## 문서

- [프로덕트 스펙](docs/product-spec.md): 사용자 문제, 제품 가설, 핵심 흐름, 성공 지표, MVP 범위
- [기술 스펙](docs/technical-spec.md): 기술 선택 기준, 1순위/2순위 비교, MVP 권장 기술 조합
- [개발 인프라](docs/development-infra.md): AWS, Google Cloud, Azure, 가성비 대안 서비스 비교와 권장 조합
- [개인정보 보호 조치](docs/privacy-compliance.md): 개인정보 분류, 동의 구조, AI 처리, 위치성 정보, 유출 대응
